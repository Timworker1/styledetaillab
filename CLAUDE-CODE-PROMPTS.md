# Claude Code — Промпты для сборки

Как пользоваться: открой Claude Code в пустой папке, **добавь `PROJECT-BRIEF.md` в проект**, затем вставляй промпты ниже **по порядку**. Не вставляй все сразу — фаза за фазой: получил результат → проверил → дальше. Каждая фаза опирается на предыдущую.

> Важно: **сайт делается на английском языке** (рынок — Дублин). Промпты ниже на русском для тебя, но весь UI-текст и копирайт сайта — на английском (приведён в кавычках). Дизайн — «Dark Emerald» из брифа §3: тёмная фотографичная база (в стиле референсов клиента Dublin Details / Detailing HQ), чёрно-белая палитра + фирменный изумрудный акцент `#1FA37A`; заголовки — жирный гротеск `Archivo` uppercase, текст — `Hanken Grotesk`. Поднять уровень относительно референсов: чище типографика, глубина, студийный свет, плавные анимации.

## Промпт 0 — Старт (вставить первым)

```
Прочитай PROJECT-BRIEF.md полностью — это единый источник правды по проекту. Делаем премиальный тёмный mobile-first лендинг для бизнеса мобильного детейлинга в Дублине, Ирландия. Язык сайта — английский, цены в евро с НДС (gross).

Дизайн строго по брифу §3 «Dark Emerald»: тёмная фотографичная база в стиле референсов клиента (Dublin Details, Detailing HQ), но чище и дороже. Палитра чёрно-белая + единственный фирменный акцент — изумруд: фон #0A0A0B, панели #141416, границы #232327, текст #FFFFFF/#F2F2F2, muted #9A9AA0, акцент #1FA37A (светлый #54C39A, тёмный #16775A). Акцент работает точечно (основная кнопка, eyebrow, активные состояния, hairline, звёзды, тонированный свет в hero) — база остаётся чёрной. Заголовки — Archivo (800/900), uppercase, плотный трекинг; текст/UI — Hanken Grotesk; оба через @import Google Fonts. НЕ используй золото, хром, серифные дисплеи и пёстрые палитры.

Сначала настрой проект, до фич:

- Framer Motion для анимаций, lucide-react для иконок
- Чистая структура: один компонент на блок страницы

- Единый редактируемый конфиг src/config/pricing.ts со всеми данными калькулятора (варианты; размеры авто с ценами, временем и примерами машин; финиши; допы; список «что входит») — точно как в брифе §5
- src/config/site.ts для названия, телефона, WhatsApp, email, соцсетей, списка районов (плейсхолдеры там, где значений нет)

- Глобально учитывай prefers-reduced-motion

Сначала собери каркас проекта и глобальный layout/тему. Покажи структуру файлов и тему до сборки отдельных секций. Блоки страницы пока не делай.
```

## Промпт 2 — Калькулятор (ядро)

```
Собери калькулятор «Configure Your Detail» (бриф §5) — самый важный блок. Всё тяни из src/config/pricing.ts. Язык — английский.

- Шаг 1: табы вариантов (Exterior / Interior / Complete)
- Шаг 2: размер авто S/M/L/XL с ценой и временем на размер + info-тултип с примерами машин
- Шаг 3: тумблер финиша Wax (включён) / Ceramic Sealant (+€60). Сохрани различие «финиш vs spray ceramic» в тексте (см. открытый вопрос в брифе)
- Боковая колонка: фикс. список «What's included» (меняется по варианту)
- Панель итога: base package, selected add-ons, estimated time, Total (incl. VAT)
- Логика: Total = base(variant×size) + finish modifier + Σ add-ons; время тоже живое
- Цены gross, без переключателя net/gross
- Главный CTA результата «Get my quote»: (a) предзаполняет форму заявки и (b) предлагает «Send on WhatsApp» с готовым сообщением (конфигурация + total + плейсхолдер Eircode)

Стеклянная (glass-morphism) панель на тёмной теме, золото на активных состояниях. Полная клавиатурная доступность. Цена/время обновляются мгновенно и плавно (анимированное число).
```

### Советы по ходу сборки
- Корми Claude Code **по одному промпту**; проверяй и правь перед следующим.
- Если фаза выдаёт слишком много за раз — попроси сделать только один блок.
- Все цифры, копирайт, районы и контакты держи в **конфиг-файлах**, чтобы правки были в одном месте.
- Меняй плейсхолдеры (логотип, видео, before/after, отзывы, контакты) по мере получения реальных ассетов.
- До финальной сборки реши **открытый вопрос «финиш Ceramic vs доп Spray Ceramic»** (бриф §5).
